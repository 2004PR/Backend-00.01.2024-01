Hackaton 07 - Programar un API Gateway que nos ayude a consultar la siguiente lista de API's:

  • Github get localhost:4000/api/github/{owner}
  • Clima get localhost:4000/api/openweather/{lat}/{lon}
  • Exchange get localhost:4000/api/exchange/{BASE}/{base}/{target}
  • Pokemon get localhost:4000/api/pokemon/{limit}
  • Pokemon Ability get localhost:4000/api/pokemon/ability/{id}
  • Rick and Morty get localhost:4000/api/rickandmorty/chacarter
  • Rick and Morty by Id get localhost:4000/api/rickandmorty/chacarter/{id}
  • Cocktail get localhost:4000/api/cocktail
  • Store get localhost:4000/api/store
  • Photo get localhost:4000/api/photo/{id}
  • NASA get localhost:4000/api/nasa/mars
