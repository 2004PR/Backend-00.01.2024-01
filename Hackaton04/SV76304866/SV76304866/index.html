¿Cómo defines una función?

Es un bloque de código que realiza una tarea específica. Puede recibir datos como entrada, procesarlos y devolver un resultado. Para definir una función, se utiliza la sintaxis en JavaScript:

function nombreDeLaFuncion(argumentos) {
      // Código a ejecutar
      }

¿Hasta cuántos argumentos puedes declarar en una función?

En cuanto al número de argumentos que se pueden declarar en una función,
no hay un límite específico. Puedes declarar tantos argumentos como necesites en una función en JavaScript.
Sin embargo, se recomienda mantener el número de argumentos lo más bajo posible para mejorar la legibilidad y la mantenibilidad del código.
