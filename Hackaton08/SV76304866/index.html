LINEA DE TIEMPO DE LAS BASES DE DATOS

  • TARJETAS PERFORADAS

    1884

    Herman Hollerith inventó la máquina automática de perforación de tarjetas perforadas, siendo nombrado así el primer ingeniero estadístico de la historia. En esa época, los censos se realizaban de forma manual.

  • CINTAS MAGNÉTICAS

    1950

    Oberlin Smith con este mecanismo se empezó a automatizar la información referente a nómina. con estas cintas magnéticas, se automatizó la información y se dio respaldos. Esto sirvió para suplir las necesidades de información de las nuevas industrias. y a través de este mecanismo se empezaron a autoatizar la información, con la desventaja de que solo se podia hacer de forma secuencial.

  • POPULARIZACIÓN DE LOS DISCOS

    1960

    El uso de computadoras se vuelve más accesible para las organizaciones privadas

  • PRIMER TÉRMINO BASE DE DATOS

    1963

    Charles Bachman supuso un nuevo tipo de Base de Datos conocido como modelo de red.

  • MODELO RELACIONAL

    1970

    Edgar Frank Codd publicó un importante documento para proponer el uso de un modelo de base de datos relacional, y sus ideas cambiaron la forma de pensar, es un modelo, el esquema de la base de datos o la organización lógica se desconecta del almacenamiento de información física, y esto se convirtió en el principio de norma para los sistemas de base de datos.

  • SDL

    1973

    Larry Ellison, Bob Miner y Ed Oates fundan una empresa de consultoría llamada Software Development Laboratories (SDL) con clave Orcle.

  • MODELO ENTIDAD RELACIÓN O ER

    1976

    Creado por Peter Chenestá, se basa en la percepción del mundo real que consta de una colección de objetos básicos, llamados entidades, y de relaciones. Este modelo le permite a los diseñadores centrarse en la aplicación de los datos en lugar de la estructura de la tabla lógica.

  • SQL

    1986

    pasó a ser el estandar ANSI y en 1987 OSI.

  • BASE DE DATOS ORIENTADO A OBJETOS

    1987

    Desde este año se desarrollaron herramientas como Excell y Acces del paquete de Microsoft Office y la creación de WWW "Word Wide Web", se facilitó la consulta de datos.

  • ACTUALIDAD

    Siglo XXI

    En la actualidad, las tres grandes compañias que dominan el mercado de las bases de datos son IBM, Micrisift y Oracle. Por su parte, en el mundo de internet, la compañia que genera la gran cantidad de inforación es Google. Aunque existen una gran variedad de software que permiten crear y manejar bases de datos con gran facilidad, como por ejemplo LINQ, que es un proyecto de Microsoft que agrega consultas nativas semejantes a las de SQL a los lenguajes de la plataforma .NET
